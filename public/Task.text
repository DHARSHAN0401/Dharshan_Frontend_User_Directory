(React.js)
**Objective:** Build a responsive single-page application that fetches and displays user data from a public API.
**Requirements & Deliverables:**
●	• Use React.js with functional components.
●	• Fetch data from 'https://jsonplaceholder.typicode.com/users'.
●	• Display the data in a styled, responsive table or card layout.
●	• Include a search bar to filter users by name.
●	• Use Material UI or CSS for styling.
**Evaluation Criteria:**
●	• Code structure and reusability – 30%
●	• API integration and functionality – 30%
●	• Responsive UI design – 20%
●	• Clean UI and creativity – 20%
